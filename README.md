Estrutura do Repositório

O projeto está organizado seguindo as boas práticas de versionamento exigidas pelo desafio, utilizando a pasta `docs/` como raiz para a publicação do GitHub Pages:

```text
├── docs/                       # Pasta principal de publicação (GitHub Pages)

│   ├── apresentacao/           # Diretório com os materiais visuais do projeto

│   └── documentacao/           # Diretório com a documentação do escopo
